‚ùå VIDEO CALLS NOT WORKING?

Problem 1: Button does nothing
-------------------------------
Solution:
1. Open browser console (F12)
2. Look for JavaScript errors
3. Try refreshing page (Ctrl+R)
4. Clear browser cache
5. Try different browser

Problem 2: Camera/Mic not working
----------------------------------
Solution:
1. Click "Allow" when browser asks for permissions
2. Check if other apps are using camera
3. Test camera at: webcamtests.com
4. Make sure HTTPS is enabled (not HTTP)

Problem 3: Can't see friend in call
------------------------------------
Solution:
1. Both must use EXACT same code
2. Both must click "Start" button
3. Check internet connection
4. Try creating new room
5. Make sure friend typed code correctly

Problem 4: Black screen in video
---------------------------------
Solution:
1. Allow camera/mic permissions
2. Close other video apps (Zoom, Teams, etc.)
3. Restart browser
4. Try incognito/private mode
5. Update browser to latest version

Problem 5: "Cannot join room" error
------------------------------------
Solution:
1. Check spelling of room code
2. Codes are case-sensitive: "Room" ‚â† "room"
3. Try new room code
4. Clear browser cache
5. Check firewall isn't blocking Jitsi
